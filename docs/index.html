Run Types

A collection of types and utils for managing dynamically typed objects at runtime

Base Type Definition

All RunType type definitions share the following structure:

  • __type string

    The name of the specific type, as defined below.

  • constraints Record<string, unknown>

    Any constraints applied to the type. See Contraints.

Native Types

Platform-specific types. Currently supported are string, number, and boolean. These represent the types native to the web platform. Adding other types should be trivial.
